# Manuelle Zertifikate? Nicht mehr.
Zertifikate laufen ab, werden übersehen oder falsch ausgestellt — und genau dann wird es teuer. Manuelle Prozesse kosten Zeit, erzeugen Fehler und erhöhen das Sicherheitsrisiko.  
ACME ADCS Server automatisiert Ausstellung und Erneuerung vollständig über das ACME Protokoll. Er verbindet die Microsoft Active-Directory Certificate Authority mit dem Automatic Certificate Management Environment (bekannt von Let’s Encrypt) und ermöglicht Ihnen, mit einem ACME (RFC 8555) kompatiblen Klienten Zertifikate Ihrer Active-Directory Certificate Authority auszustellen.  
So behalten Sie jederzeit Kontrolle und Sicherheit, ohne zusätzlichen Aufwand.
